数学的準備

以下の常微分方程式

ddtx(t)=f(x(t))\begin{aligned} \frac{d}{dt} \bm{x}(t) = \bm{f}(\bm{x}(t)) \end{aligned}dtd​x(t)=f(x(t))​

に対して，シミュレーションF:Rk→Rk\bm{F}: \mathbb{R}^k \rightarrow \mathbb{R}^k F:Rk→Rk を用いて，

xt=F(xt−1)\begin{aligned} \bm{x}_t = \bm{F}(\bm{x}_{t-1}) \end{aligned}xt​=F(xt−1​)​

による時間発展の記述を考える．

数値実験

ここでは，簡単に以下の連立常微分方程式

dx1dt=μx1dx2dt=λ(x2−x12)\begin{aligned} \frac{dx_1}{dt}& = \mu x_1 \\ \frac{dx_2}{dt}& = \lambda (x_2 - x_1^2) \end{aligned}dtdx1​​dtdx2​​​=μx1​=λ(x2​−x12​)​

に対して適用を試みる．ただし，μ=−0.1\mu = -0.1μ=−0.1，λ=−1\lambda = -1λ=−1 と設定する．

# --- 1. システムダイナミクスの定義 (変更なし) ---
const μ = -0.1f0
const λ = -1.0f0

function simple(x)
    dx1 = μ * x[1]
    dx2 = λ * (x[2] - x[1]^2)
    return [dx1, dx2]
end
